# 기호교육 등급 변환기

9등급 ↔ 5등급 변환을 위한 테이블 기반 웹 애플리케이션

① 표와 일치하는 입력 값을 우선 조회, 
② 일치하는 값이 없을 경우 인접한 두 행으로 선형 보간

## 기능

- 9등급 → 5등급 변환
- 5등급 → 9등급 변환
- 결과 값 출력: 상 / 평균 / 하
- 표에서 입력 값과 대응하는 행 하이라이트
- 화면 가로/세로 분할 레이아웃 전환

## 입력 규칙

| 모드 | 입력 범위 |
|-----|-----|
| 9등급 → 5등급 | 1.00 ~ 9.00 |
| 5등급 → 9등급 | 1.00 ~ 5.00 |

입력값은 소수점 둘째 자리까지 허용
